This plugin will update hyperlinks in articles are per options set in the plugin settings page.

== Description ==

- To use this plugin, you must install it and activate in the Plugins page.
- After that Go to `Settings -> Custom hyperlinks` page and set options as per your requirements.
- Confirm the preview of update hyperlinks in `Preview` tab.
- Toggle the `Apply` switch in the `Preview` tab.
- That's it.
